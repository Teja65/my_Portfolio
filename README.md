# Portfolio — Konakalla Teja Venkatesh

Personal portfolio built with React, Vite, and TypeScript.

## Run locally

```bash
npm install
npm run dev
```

## Add your assets

1. **Profile photo** — place your image at `public/profile/photo.jpg`
2. **Resume** — place your PDF at `public/resume/resume.pdf`

## Build for production

```bash
npm run build
npm run preview
```

## Deploy

Deploy the `dist` folder to [Vercel](https://vercel.com), Netlify, or GitHub Pages.
